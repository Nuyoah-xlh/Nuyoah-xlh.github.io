什么是Linux？

​ Linux一般用来做服务器端的操作系统。

什么是服务器端？

  • 所有的网站，所有的网络游戏都需要服务器端

  • 一半以上的家用电脑使用Windows，但是90%以上的服务器端使用Linux

Linux的应用

  • 学习了Linux可以做Linux运维（运行和维护）
  • 学习了Linux可以做嵌入式开发
  • 服务器端开发

Linux版本选择&系统下载安装

版本选择

Linux版本分为：

​ 内核版本：操作系统心脏，处理跟各种硬件打交道的工作。

​ 发行版本（内核版本+各种应用软件（办公 媒体播放 浏览器 数据库 …. ））

我们一般选择Ubuntu

虚拟机安装

  1. 安装Vmware虚拟机，前往下载，选择如下版本：
  2. 根据自己电脑的信息选择合适的版本下载即可。
  3. 下载后直接安装，可选择安装的位置。再进行账号注册绑定等即可安装成功。（过程中需要的密钥，可以上网自行查找）

Ubuntu安装

  1. 前往Ubuntu官网下载系统文件，点击传送
  2. 下载完成后，进入Vmware虚拟机，创建新虚拟机，找到下载的.iso文件并根据提示进行创建（考虑到兼容性，建议选择语言为英语）
  3. 填写个人信息，密码等要素
  4. 等待安装成功。

Linux使用

文件系统

Linux文件系统（目录和文件）

  • Windows下有盘符
  • Linux没有盘符的概念，只有一个根目录

Terminal

​ 一般常使用命令行来对服务器进行操作，Linux上的命令行是一个名为Terminal的应用。

常用命令

  • pwd

    • print work directory查看所在路径

  • ls

    • list 查看当前目录下的内容（文件和文件夹）
    • ls -l （以列表形式展现）
    • ls -a （显示隐藏目录）
    • ls -l -h （列表的同时显示大小）
    • ls -lha （组合使用）
    • 星号通配符在ls中的使用（*代表0个或0个以上的任意字符）
    • 问号通配符在ls中的使用（?表示一个任意字符）
    • 通配符的括号用法

  • cd

    • change directory打开目录，进入到当前目录下的某个文件夹
    • cd ..表示前往上一级路径
    • cd ./ 当前目录
    • cd ~ 回到home

  • clear

    • 清空屏幕

常用快捷键

  • tab自动补全文件名功能

    • 会从当前文件目录下查找匹配的文件名进行匹配，若无匹配，则无反应；按两次可显示多个匹配项。

  • 方向键上

    • 到上一条命令

  • 方向键下

    • 到下一条命令
